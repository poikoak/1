1. Пользователь вводит путь к исходной папке, программа копирует все файлы из исходной
папки в результирующую папку, которую тоже указывает пользователь.

2. Для первой задачи обрабатывать подпапки исходной папки.

3. Разработать командный интерпретатор (по типу cmd), который понимает следующие команды:
- cls - очистка экрана
- exit - выход из программы
- dir - показать содержимое текущей папки
- type - показать содержимое файла (пример: type c:\temp\text.txt)
