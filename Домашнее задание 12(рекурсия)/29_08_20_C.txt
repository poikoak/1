1. Пользователь вводит 5 чисел, рекурсивная функция подсчитывает сумму отрицательных чисел.

2. Пользователь вводит числа, пока не введёт 0. Рекурсивная функция подсчитывает
количество троек среди введенных чисел.

Пример:
3
2
4
5
3
1
3
0

Ответ: 3

3. Пользователь вводит 5 чисел, рекурсивная функция возвращает 1, если все числа положительные, а
иначе 0

4. Пользователь вводит 5 чисел, рекурсивная функция возвращает 1, если все числа идут в порядке
возрастания, а иначе 0

5. Пользователь вводит 5 чисел, рекурсивная функция возвращает 1, если все числа являются
простыми, а иначе 0 (выяснение простоты тоже можно сделать рекурсивным)
